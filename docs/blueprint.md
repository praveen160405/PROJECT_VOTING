# **App Name**: VerityVote

## Core Features:

- Voter Authentication: Secure multi-factor authentication including Voter ID, password, facial recognition, and OTP verification.
- Voter Registration: Secure form to upload ID proof, capture live face image, and hash voter data before storing the identity hash on the blockchain.
- Vote Casting: Display of a verified candidate list and secure vote casting using smart contracts.
- Admin Panel: Secure admin login to verify voters, manage candidates, and control the election.
- Results Dashboard: Real-time vote count displayed in charts and graphs.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and security.
- Background color: Light gray (#E0E0E0), a desaturated version of the primary, to ensure legibility.
- Accent color: Teal (#009688) to highlight key actions and information.
- Body and headline font: 'Inter', a sans-serif font that offers a modern, neutral and machined look, makes it suitable for headlines and body text
- Modern glassmorphism or material UI design for login page with smooth transitions and responsive design (mobile + desktop).
- Subtle animations on vote submission and result updates.
